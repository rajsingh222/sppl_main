Overview

Introduce precision and reliability to your structural health monitoring with Digitex’ xCAS. Our high-performance system offers multi-channel real-time data acquisition and structural analysis with manual and event-driven triggering. The system can integrate most sensor types to offer a more detailed picture of a structure’s health. With a sampling rate of up to 1,000 samples per second, the xCAS system features an extensive set of remote tools for real-time monitoring. The Voyager® central acquisition software, integrated into xRover, provides Real-Time data management and analysis. User can access the data from any internet enabled computer with appropriate login information. The system is designed to work with one or more xRover units and can integrate and manage almost all sensor types. Advanced analysis can be done using Digitex’ powerful data analysis software Observer®.
